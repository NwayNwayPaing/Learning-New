Using R function:p_load

When we want to use many functions from library, we would open each function like below.

library(table1) #provides table
library(ggplot2) #provides graph
library(MASS) #test

Instead, we can use “pacman” package to open many functions at once.

First, install the package using the codes below.

installed.packages("pacman")
##      Package LibPath Version Priority Depends Imports LinkingTo Suggests
##      Enhances License License_is_FOSS License_restricts_use OS_type Archs
##      MD5sum NeedsCompilation Built

Then, we will run the p_load function.

pacman::p_load(table1,ggplot2,Mass)
